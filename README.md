## Introduction

- Checkers game build on top of Cosmos SDK


## Prerequisites

- Go 1.21+
- Cosmos SDK
- Ignite CLI

## Installation

1. Clone the repository:
```bash
git clone https://github.com/pri-3x/checker-minimal.git
cd checker-minimal
```

2. Install dependencies:
```bash
go mod tidy
```
3. Build Protogen
```bash
make proto-gen
```
